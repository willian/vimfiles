# Vimfile

## Installation

1. `git clone git@github.com:willian/vimfiles.git`
2. `cd vimfiles`
3. `sh install.sh`
4. Enjoy! =)

## Compatibility

### Mac

* Vim
* MacVim

### Linux

* Vim
* gVim
